# Sejong_OS_
OpenSource Project in Sejong University

# 개발 목적
기존 검색 엔진의 광고와 파워링크등을 제외하여 검색한 결과를 보여주기 위한 프로젝트

# 개발 스택
1. back : express, cheerio 사용
2. front : ejs와 vanilla.js 사용
3. 디자인 : Adobe XD, Adobe Photoshop 사용
4. 호스팅 : Amazon EC2 사용

# 기능 구현
1. 실시간 검색어 크롤링
2. 검색어 1개로 3개의 웹사이트 동시 크롤링
3. 개별 페이지 기능 추가

# 기능 결과
1. 실시간 검색어 구현 완료
2. 동시 검색 기능 구현 완료
3. 페이지별 기능 구현 완료

# 현재 상황
